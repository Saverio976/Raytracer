void* getCreator ( void  )

Get the creator.

The creator must be a class that has a .create(const ISetting &settings) method. This method must create an [ ILight | ICamera | IPrimitive | IFilter ]. There is no verification that the returning value is ok. So be cautious. Thx. For [ ILight | ICamera | IPrimitive ] they must inherite from IEntityCreator. For [ IFilter ] they must inherite from IFilterCreator.

Returns
the creator
