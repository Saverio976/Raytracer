Detailed Description

Logger singleton.

LogLevel: -1 - no messages 0 - only fatal are displayed 1 - error and fatal are displayed 2 - warn, error and fatal are displayed 3 - info, warn, error and fatal are displayed 4 - debug, info, warn, error and fatal are displayed 5 - trace, debug, info, warn, error and fatal are displayed

Attention: debug and trace will only do something in debug mode compilation. (set(CMAKE_BUILD_TYPE Debug) in CMakeLists.txt)
