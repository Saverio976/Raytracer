\doxysubsection{Detailed Description}
\mbox{\hyperlink{a01406}{Logger}} singleton. 

Log\+Level\+: -\/1 -\/ no messages 0 -\/ only {\ttfamily fatal} are displayed 1 -\/ {\ttfamily error} and {\ttfamily fatal} are displayed 2 -\/ {\ttfamily warn}, {\ttfamily error} and {\ttfamily fatal} are displayed 3 -\/ {\ttfamily info}, {\ttfamily warn}, {\ttfamily error} and {\ttfamily fatal} are displayed 4 -\/ {\ttfamily debug}, {\ttfamily info}, {\ttfamily warn}, {\ttfamily error} and {\ttfamily fatal} are displayed 5 -\/ {\ttfamily trace}, {\ttfamily debug}, {\ttfamily info}, {\ttfamily warn}, {\ttfamily error} and {\ttfamily fatal} are displayed

Attention\+: {\ttfamily debug} and {\ttfamily trace} will only do something in debug mode compilation. ({\ttfamily set(\+CMAKE\+\_\+\+BUILD\+\_\+\+TYPE Debug)} in {\ttfamily CMake\+Lists.\+txt})
